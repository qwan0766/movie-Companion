# 项目指导手册：基于 LangGraph 的腾讯视频智能观影助手实习项目

**项目名称**  
基于 LangGraph 的腾讯视频智能观影助手实习项目

**核心技术栈**
- **工作流编排**：LangGraph（StateGraph + Nodes + Edges + Checkpoint）
- **LLM 集成**：LangChain + LangGraph
- **API 服务**：FastAPI（RESTful 接口）
- **存储**：SQLite（用户历史、基础数据） + Chroma（向量数据库） + Neo4j（影视知识图谱）
- **追踪调试**：LangSmith
- **前端界面**：Streamlit Web 界面
- **LLM 模型**：优先使用开源/本地模型（Llama3 / Qwen / 通义千问），也可切换 GPT-3.5 进行对比

**项目目标**  
解决腾讯视频用户“**找片难、疑问多、计划乱**”三大痛点，提供**智能观影推荐、影视咨询、观影计划制定**三位一体的智能助手服务。

**技术架构（分层明确）**

1. **用户界面层**：Streamlit Web 界面（对话 + 推荐卡片 + 用户反馈）
2. **API 服务层**：FastAPI（封装 LangGraph 工作流，提供 `/chat`、`/recommend`、`/plan` 等 RESTful 接口）
3. **智能体层（LangGraph 工作流 - 状态机管理）**：
    - 用户意图识别 Agent
    - 视频检索 Agent（向量数据库 + 传统过滤）
    - 知识查询 Agent（Neo4j 知识图谱 + RAG）
    - 推荐生成 Agent
    - 对话管理 Agent
4. **数据层**：
    - 腾讯视频数据库（模拟数据）
    - 向量数据库（Chroma，支持语义相似检索）
    - 影视知识图谱（Neo4j）
    - 用户历史记录存储（SQLite）

**技术目标**
- 掌握 LangGraph 框架构建多 Agent 状态机工作流的能力
- 理解 RAG（检索增强生成）在垂直领域（影视）的实际应用
- 实践腾讯视频真实业务场景下的智能助手开发
- 培养工程化思维和全栈 AI 应用开发能力

**最终交付物（必须全部完成）**
1. 完整可运行的智能观影助手原型（本地一键启动）
2. 完整技术文档 + 工作流程图（Mermaid / draw.io）
3. 测试报告（单元测试 + 集成测试 + 端到端测试）
4. 学习笔记 + 技术调研报告（含踩坑记录、性能数据、优化思路）

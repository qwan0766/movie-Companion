### 2.2 Pydantic 模型

```python
class ChatRequest(BaseModel):
    query: str
    thread_id: str = "default"

class MultiTurnRequest(BaseModel):
    messages: list[dict]
    thread_id: str = "default"

class ChatResponse(BaseModel):
    user_intent: str
    intent_confidence: float
    response: str
    retrieved_videos: list = []
    knowledge_result: dict = {}
    plan: dict = {}
    errors: list = []
    thread_id: str = "default"
```

### 2.3 错误处理

```python
class APIError(BaseModel):
    detail: str
    error_code: str

# 统一异常处理器
@app.exception_handler(Exception)
async def global_exception_handler(request, exc):
    return JSONResponse(
        status_code=500,
        content={"detail": str(exc), "error_code": "INTERNAL_ERROR"},
    )
```

## 四、路由实现

### 4.1 单轮对话 `/chat`

```python
@router.post("/chat", response_model=ChatResponse)
async def chat(request: ChatRequest):
    """单轮对话"""
    try:
        result = run_query(request.query, thread_id=request.thread_id)
        return ChatResponse(
            user_intent=result.get("user_intent", ""),
            intent_confidence=result.get("intent_confidence", 0.0),
            response=result.get("response", ""),
            retrieved_videos=result.get("retrieved_videos", []),
            knowledge_result=result.get("knowledge_result", {}),
            plan=result.get("plan", {}),
            errors=result.get("errors", []),
            thread_id=request.thread_id,
        )
    except Exception as e:
        return JSONResponse(
            status_code=500,
            content={"detail": str(e), "error_code": "CHAT_ERROR"},
        )
```

### 5.1 main.py 整合

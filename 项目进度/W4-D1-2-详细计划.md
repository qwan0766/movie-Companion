## 四、后端对接（api_client.py）

```python
import httpx
import streamlit as st

API_BASE_URL = "http://localhost:8000"

@st.cache_data(ttl=60)
def health_check() -> dict:
    resp = httpx.get(f"{API_BASE_URL}/health")
    return resp.json()

def send_chat(query: str, thread_id: str = "default") -> dict:
    resp = httpx.post(
        f"{API_BASE_URL}/chat",
        json={"query": query, "thread_id": thread_id},
        timeout=30,
    )
    return resp.json()
```

## 五、状态管理

使用 `st.session_state` 管理前端状态：

| key | 类型 | 说明 |
|-----|------|------|
| `messages` | list[dict] | 对话历史 `[{"role": "user"/"assistant", "content": ..., "data": ...}]` |
| `thread_id` | str | 当前会话 ID |
| `loading` | bool | 是否正在请求 |
| `response_data` | dict | 最近一次响应完整数据 |

## 二、模块说明

### 2.1 Agent 模块 (`agents/`)

| Agent | 功能 | 路由 | 方法 |
|-------|------|------|------|
| `IntentAgent` | 用户意图分类 | 入口节点 | 规则匹配（正则+权重） |
| `RetrievalAgent` | 视频找片推荐 | find_movie | 混合检索 |
| `KnowledgeAgent` | 影视知识查询 | ask_info | Neo4j/SQLite |
| `PlanAgent` | 观影计划制定 | make_plan | 检索+编排 |
| `ChatAgent` | 闲聊对话 | chat/unknown | 模板匹配 |

所有 Agent 继承自 `BaseAgent`，实现 `process(state) -> dict`。

### 2.2 工作流模块 (`graph/`)

| 文件 | 功能 |
|------|------|
| `state.py` | AgentState 定义（10 个字段） |
| `nodes.py` | 5 个节点函数 + safe_node 包装 |
| `graph.py` | StateGraph 构建 + 编译 + 条件边 + MemorySaver |

**路由逻辑：**

```
START → intent_agent
         ├── find_movie → retrieval_agent
         ├── ask_info   → knowledge_agent
         ├── make_plan  → plan_agent
         ├── chat       → chat_agent
         └── unknown    → chat_agent
所有 Agent → respond_node → END
```

### 2.3 工具模块 (`tools/`)

| 文件 | 功能 |
|------|------|
| `search_tools.py` | parse_query / semantic_search / filter_videos / hybrid_search |
| `knowledge_tools.py` | knowledge_search / detect_query_type（Neo4j→SQLite 降级） |
| `tool_definitions.py` | 4 个 LangChain @tool 包装 |

### 2.4 API 模块 (`api/`)

| 路由 | 方法 | 功能 |
|------|------|------|
| `/health` | GET | 健康检查 |
| `/chat` | POST | 单轮对话 |
| `/chat/multi` | POST | 多轮对话 |
| `/workflow/mermaid` | GET | 工作流流程图 |

### 2.5 数据层

| 存储 | 用途 | 技术 |
|------|------|------|
| SQLite | 视频/演员/导演/分类 关系数据 | `db/sqlite_db.py` |
| Chroma | 视频语义向量检索 | `db/chroma_db.py` |
| Neo4j | 影视知识图谱（可选） | `knowledge_graph/` |

## 三、数据流

```
用户输入 "推荐科幻电影"
  → POST /chat {query, thread_id}
  → run_query()
  → StateGraph.invoke()
    → intent_agent.process()
      → _classify_intent() → "find_movie" (0.85)
    → route_after_intent() → "retrieval_agent"
    → retrieval_agent.process()
      → parse_query("推荐科幻电影") → {genre: "科幻"}
      → hybrid_search("推荐科幻电影")
        → semantic_search() [Chroma]
        → filter_videos(genre="科幻") [SQLite]
        → 融合去重排序
      → _build_response() → 推荐语
    → respond_node()
      → 收集 response
  → 返回 {user_intent, retrieved_videos, response}
  → 前端渲染：左侧对话气泡 + 右侧卡片网格
```

## 六、扩展指南

### 添加新 Agent

1. 在 `agents/` 下创建新 Agent 类，继承 `BaseAgent`
2. 实现 `process(state) -> dict` 方法
3. 在 `graph/nodes.py` 中注册节点函数
4. 在 `graph/graph.py` 中添加节点和边
5. 在条件边路由映射中添加新意图
6. 在 `agents/intent_agent.py` 中添加意图识别规则
7. 编写测试用例

### 添加新 Tool

1. 在 `tools/tool_definitions.py` 中使用 `@tool` 装饰器定义
2. 添加到 `TOOL_REGISTRY` 和 `TOOL_MAP`
3. 在对应 Agent 的 `process()` 中调用

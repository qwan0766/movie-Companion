# 腾讯视频智能观影助手 — 用户手册

> 版本 v0.1 | 2026-04-29

---

## 一、简介

腾讯视频智能观影助手是一个基于大语言模型的多 Agent 智能助手，帮助你解决观影过程中的三大痛点：

- **找片难** — 模糊描述也能精准推荐
- **疑问多** — 演员、导演、影视知识随问随答
- **计划乱** — 自动编排观影计划

---

## 二、启动指南

### 2.1 环境要求

- Python 3.10+
- 操作系统：Windows / macOS / Linux

### 2.2 启动步骤

**第一步：安装依赖**

```bash
cd tencent_video_agent
pip install -r requirements.txt
```

**第二步：配置环境变量**

```bash
cp .env.example .env
```

编辑 `.env`，至少需要配置 LangSmith API Key（可选，不配不影响核心功能）。

**第三步：初始化数据库**

```bash
python db/init_db.py
```

**第四步：启动 API 服务**

```bash
python main.py
```

终端显示 `API 文档: http://0.0.0.0:8000/docs` 即启动成功。

**第五步：打开前端界面**

打开浏览器访问 **http://localhost:8501**，或在新终端执行：

```bash
streamlit run frontend/app.py
```

## 三、功能使用

### 3.1 找片推荐

输入模糊描述，助手自动理解你的需求：

| 你可以这样说 | 示例 |
|-------------|------|
| 按类型 | "推荐科幻电影" "有什么好看的喜剧" |
| 按演员 | "找张毅演的科幻片" |
| 按评分 | "高分的剧情片" |
| 按年代 | "2020年以后的悬疑片" "近三年的国产剧" |
| 组合查询 | "找2020年后高分国产科幻片" |

### 3.2 影视知识查询

查询演员、导演、视频的详细信息：

| 你可以这样说 | 示例 |
|-------------|------|
| 演员作品 | "介绍一下张毅" "周星驰演过哪些电影" |
| 导演作品 | "谁执导了流浪地球" |
| 视频详情 | "介绍星际穿越" |
| 谁演的 | "让子弹飞谁演的" |

### 3.3 观影计划制定

让助手为你编排观影计划：

| 你可以这样说 | 示例 |
|-------------|------|
| 今晚计划 | "帮我规划今晚看什么" |
| 周末计划 | "帮我规划周末看什么" |
| 按心情 | "推荐轻松搞笑的，周末看" |

### 3.4 闲聊互动

简单的问候和感谢：

| 你可以这样说 | 示例 |
|-------------|------|
| 问候 | "你好" "嗨" |
| 感谢 | "谢谢" |
| 告别 | "再见" |

## 四、常见问题

**Q：前端页面打不开？**
确认 API 服务已启动（`python main.py`），然后刷新前端页面。

**Q：搜索没有结果？**
试试调整描述，加上更具体的类型、年代或演员名。

**Q：可以搜索真实腾讯视频的内容吗？**
当前版本使用模拟数据（100 部视频、60 位演员），用于功能演示。

**Q：如何切换对话？**
点击右上角的「新对话」按钮即可开始新会话。

## 五、注意事项

1. 首次使用需要先初始化数据库（`python db/init_db.py`）
2. API 服务和前端需要同时运行（两个终端窗口）
3. 前端显示"后端未连接"，请检查 API 服务是否在 8000 端口运行
